== Description ==

Put a completely customized navigation on any page. The navigation can be set up as a contact email navigation, or even be used to create WordPress posts for front-end user submission.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the 'mp-stacks-navigation’ folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Build Bricks under the “Stacks and Bricks” menu. 
4. Publish your bricks into a “Stack”.
5. Put Stacks on pages using the shortcode or the “Add Stack” button.
